    Drush and Features-Export

    When at the command line with drush making features keep in mind you may need to use quotes. Example drush features-export --destination="sites/all/modules/custom" behat_site_settings dependencies:behat menu_custom:main-menu menu_links:main-menu:"<front>" menu_links:main-menu:"user/login" menu_links:main-menu:user menu_links:navigation_:"behat/testarea"  Without the quotes around the menu links you would get an error from bash.
